# 🐍 Snake Game

Classic Snake game built with vanilla HTML5 Canvas and JavaScript.

## 🎮 Play

**Live:** [http://ai-game-snake-855647970705.s3-website-us-west-2.amazonaws.com](http://ai-game-snake-855647970705.s3-website-us-west-2.amazonaws.com)

## Features

- Classic Snake gameplay
- Smooth neon-styled graphics
- Score tracking with local best score
- Progressive speed increase
- Keyboard controls (Arrow Keys / WASD)
- Mobile touch support with swipe and D-pad
- Pause (Space) and Restart (Enter)
- Responsive design

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| Space | Pause |
| Enter | Restart |

On mobile: swipe or use the on-screen D-pad.

## Tech Stack

- Pure HTML5 / CSS3 / JavaScript
- Canvas API for rendering
- No dependencies
- Hosted on AWS S3 (static website)

## Local Development

Just open `index.html` in a browser. No build step needed.
